Project objective

The focus in this project is to study the process since the problem understanding until the deployment of the model.

Steps of this project

Problem understanding

Deep understanding of the studied problem, answering questions like:

  • What possibly is causing the churn?
  • How churn affects the company?
  • What standards can be seen in the historical data?

Data gathering

Collect relevant data from customers that will be used to create the model

Pre-processing and data cleaning

Clean and format the data to remove bias and improve the model performance.

Example of cleaning techniques:

  • Fill null values
  • Remove duplations
  • Normalize data

Exploratory analysis

Exploratory analysis made to discover standards, distribution and bond between the data sources.

Modeling with RandomForest

Use of the RandomForest algorithm to develop the model.

Model evaluation

It's used to test the model created with new data to evaluate the precision. Tunning can be made with the results of this step.

Deployment

Develop a strategy to deploy the model that will be used with new data to make predictions.

The problem

Churn in Machine Learning context is the predction of when a customer will stop to use a service or stop to buy prodcts from a company.

You must develop a way to know it without the need of open the box.

Available informations

  • Age: Age of the customer.
  • MonthUse: How much the customer use of the band.
  • Tier: What license tier the customer has hired.
  • Satisfaction: What score the customer answered in a satisfaction pool.
  • Time: How long this customer is a customer.
  • MonthPayment: How much does the customer pay each month.
